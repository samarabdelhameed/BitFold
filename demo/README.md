# Demo Materials

This directory contains materials for presenting BitFold to the judging committee.

## Files

- `demo-script.md`: Script for 3-minute demo video
- `slides.pptx`: Presentation slides (to be created)
- `demo-recording.mp4`: Final demo video (to be recorded)
- `screenshots/`: Screenshots of the application

## Recording Instructions

1. Prepare the application:
   - Deploy canisters locally or to testnet
   - Have test data ready
   - Ensure all features work

2. Record the demo:
   - Follow the script in `demo-script.md`
   - Keep it under 3 minutes
   - Show key features clearly
   - Use screen recording software

3. Edit and finalize:
   - Add captions if needed
   - Ensure audio is clear
   - Export as `demo-recording.mp4`
