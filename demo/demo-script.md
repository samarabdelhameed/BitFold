# BitFold Demo Script (3 minutes)

## Introduction (30 seconds)
- Welcome to BitFold
- Problem: Bitcoin holders want liquidity without selling
- Solution: Use Bitcoin UTXOs and Ordinals as collateral to borrow ckBTC

## Demo Flow (2 minutes)

### 1. Landing Page (10 seconds)
- Show landing page
- Highlight key features: Deposit, Borrow, Repay, Withdraw

### 2. Deposit UTXO (30 seconds)
- Navigate to Deposit page
- Enter Bitcoin UTXO details:
  - Transaction ID
  - Output index
  - Amount
  - Address
- Show Ordinal support (if applicable)
- Submit and show success

### 3. Borrow ckBTC (30 seconds)
- Navigate to Borrow page
- Select deposited UTXO as collateral
- Show maximum borrowable (50% LTV)
- Enter borrow amount
- Submit and show loan created

### 4. Dashboard (20 seconds)
- Show loan dashboard
- Display active loans
- Show collateral status
- Show repayment progress

### 5. Repay Loan (20 seconds)
- Navigate to Repay page
- Select active loan
- Show remaining debt
- Enter repayment amount
- Submit and show success

### 6. Withdraw Collateral (10 seconds)
- Navigate to Withdraw page
- Select fully repaid collateral
- Withdraw and show success

## Conclusion (30 seconds)
- Recap: Full cycle demonstrated
- Key features:
  - Bitcoin UTXO collateral
  - Ordinals NFT support
  - Decentralized on Internet Computer
  - Secure and transparent
- Thank you!
